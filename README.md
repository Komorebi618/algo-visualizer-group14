# algo-visualizer-group14
算法过程可视化系统，基于 HTML、CSS、JavaScript 和 Canvas API 实现，包含快速排序、Dijkstra 最短路径和汉诺塔三个算法可视化模块。

## 项目基础信息介绍

### 基本要求

1. 算法选择：3个算法
   - 快速排序：柱状图交换动画
   - Dijkstra最短路径：节点图逐步染色
   - 汉诺塔：柱子加盘子
提示：项目靠考虑算法难度和Canvas显示难度
2. 使用统一入口和统一界面风格
3. 数据：每个算法支持手动输入数据或随机生成测试数据，至少2个测试用例，项目总用例不少于6条
4. 显示：
   1. 当前的输入数据
   2. 执行过程以及其中的关键状态变化、每一步的说明
   3. 最终执行结果
   4. 时间复杂度和空间复杂度说明
5. 文稿：提交 `README.md`，说明系统运行方式、算法选择方式、输入格式和测试方式。

### 拓展需求（选做）

> 当前暂不考虑，可保留后续完成的接口

1. 增加第4种类型算法
2. 展示控制功能：单步执行、自动播放、暂停、重置等基本控制；上一步回退
3. 导出算法日志
4. 同类算法对比
5. 用户保存自定义测试数据
6. 错误输入提示与输入范围检查
7. 题目讲解，关键步骤解释文字
